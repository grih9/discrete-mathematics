# discrete-mathematics
<h2>Алгоритмы для дискретной математики</h2>

<li><strong>deykstra - Алгоритм Дейкстры </strong> - находит кратчайшие пути от одной из вершин графа до всех остальных;
<li><strong>prima - Алгоритм Прима </strong> - построение минимального остовного дерева взвешенного связного неориентированного графа;
<li><strong> uorshil - Алгоритм Флойда-Уоршилла </strong> - нахождение кратчайших расстояний между всеми вершинами взвешенного ориентированного графа.
